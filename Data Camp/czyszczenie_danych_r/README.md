# Data Camp: Data Engineering & Cleaning

This folder contains certifications related to data preprocessing, manipulation, and functional programming in R and Python.

### Key Skills & Certifications:
* **Introduction to Python** – Core programming concepts for data science.
* **Data Manipulation with dplyr** – Advanced data transformation in R.
* **Cleaning Data in R** – Identifying and fixing missing or inconsistent data.
* **Reshaping Data with tidyr** – Converting data between wide and long formats.
* **String Manipulation & Regex** – Handling text data using `stringr` and Regular Expressions.
* **Working with Dates and Times** – Managing temporal data using `lubridate`.
